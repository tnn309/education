# Education for E

## Mô tả
Education là một ứng dụng web được phát triển bằng ASP.NET Core với .NET 9.0, thiết kế để quản lý các hoạt động giáo dục một cách hiệu quả. Hệ thống hỗ trợ các vai trò người dùng khác nhau (Admin, Nhân viên, Phụ huynh/Học sinh) và sử dụng PostgreSQL làm cơ sở dữ liệu. Dự án cung cấp các tính năng như quản lý người dùng, đăng ký hoạt động, xử lý thanh toán, lập lịch trình, theo dõi kết quả học tập, gửi thông báo, và tạo báo cáo, hướng đến việc tối ưu hóa quy trình giáo dục và nâng cao trải nghiệm người dùng.

## Tính năng chính
- Quản lý người dùng (Users) với các vai trò đa dạng.
- Tạo và quản lý hoạt động giáo dục (Activities).
- Hỗ trợ đăng ký (Registrations) và thanh toán (Payments).
- Gửi thông báo cá nhân hóa (Notifications).
- Lưu trữ và quản lý bình luận/dánh giá (Comments).
- Theo dõi tương tác người dùng (InteractionLogs).
- Phân công nhân viên (StaffAssignments) cho các hoạt động.
- Quản lý lịch trình (Schedules) chi tiết.
- Lưu trữ và cập nhật kết quả học tập (Results).
- Hỗ trợ tin nhắn hai chiều (Messages).
- Quản lý thông tin trung tâm (CenterInfo).
- Ghi lại lịch sử truy cập (AccessLogs).
- Tạo báo cáo (Reports) dạng JSONB.

## Yêu cầu hệ thống
- **.NET SDK**: Phiên bản 9.0 hoặc cao hơn.
- **PostgreSQL**: Phiên bản 17.
- **Công cụ dòng lệnh**: `dotnet` (đã cài đặt).
- **IDE**: Visual Studio, Visual Studio Code, hoặc tương đương.

## Cài đặt

### 1. Clone repository 
### 2. dependencies : dotnet restore
### 3. update data  : dotnet ef database update
### 4. build Project: dotnet build
### 5. end          : dotnet run
